# ERGO Versicherungsplattform

## Übersicht
Eine professionelle Lead-Generierungs-Plattform für ERGO Versicherungen mit optimierten Landing Pages und Admin-Dashboard.

## Projekt-Architektur

### Frontend (React + TypeScript)
- **Homepage** - Übersicht aller Versicherungen mit Trust-Elementen
- **Versicherungs-Landing-Pages** - Dedizierte Seiten für jede Versicherung
- **Lead-Funnel** - 3-stufiger Conversion-Funnel mit Fortschrittsanzeige
- **Admin-Dashboard** - Content Management und Lead-Verwaltung

### Backend (Express + PostgreSQL)
- **API Routes** - RESTful API für Leads und Content
- **Datenbank** - PostgreSQL mit Drizzle ORM
- **Session Management** - Express-Session für Admin-Authentifizierung

### Tracking & Analytics
- Google Analytics 4 Integration (vorbereitet)
- Meta Pixel Integration (vorbereitet)
- Event-Tracking für alle wichtigen Interaktionen
- Conversion-Tracking für Lead-Generierung
